# Pure Rust Radio Playback Stack

Lum's `radio` subcommand reimplements `ruv` with a pure Rust playback stack: `reqwest` for fetching direct HTTPS audio streams, `symphonia` for decoding, `cpal` for cross-platform audio output, `crossterm` for terminal controls, `ringbuf` for the decoder-to-audio bridge, and `rubato` for resampling. We are deliberately dropping `ffmpeg` as a runtime dependency because the current built-in station set uses MP3/AAC-LC streams that Symphonia supports. The trade-off is narrower stream compatibility than ffmpeg: v1 supports direct audio streams only, requires new built-in stations to be Symphonia-compatible, supports resampling and mono-to-stereo conversion for existing ruv stations, and does not support HE-AAC, HLS, or playlists until a real station requires them.
